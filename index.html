PREGUNTAS FRECUENTES

¿Para qué sirve la etiqueta "details"?

Es utilizado para crear un widget interactivo que el usuario puede abrir y cerrar. Es especialmente útil para mostrar información adicional que no necesita estar visible de inmediato, mejorando la usabilidad y organización del contenido en una página web.

¿Para qué sirve la etiqueta "summary"?

Define un encabezado visible para el contenido del elemento "details". Este encabezado es lo que el usuario verá y en lo que hará clic para expandir o contraer el contenido. Debe ser el primer elemento dentro de un "details". Actúa como el título o resumen del contenido oculto.
